DESCRIPTION

Juan Garcia is a self-taught technologist from California passionate about Linux, open-source infrastructure, and ethical computing. With a background in IT support and systems administration, Juan builds internal tooling and networked environments that emphasize reliability, security, and thoughtful design.

Driven by curiosity and purpose, Juan explores the intersection of automation, homelab architecture, and cybersecurity while maintaining a long-term vision of building tools and infrastructure that uplift small teams and communities. He believes in learning deeply, building deliberately, and documenting the journey.

WHAT I WORK ON

  • gg3utils – A collection of personal utilities for Python and Bash automation. Includes logging, file sync, system checks, and tool wrappers for DevOps workflows.
  • nginx-hardened – Hardened NGINX deployment template with strong SSL config, rate limiting, and secure headers.
  • jenkins-pipelines – Jenkins jobs to automate snapshotting VMs, syncing backups, and restarting critical services.
  • ssh-key-structure – A personal SSH key strategy using namespacing (`key.gg3.git`) for scalable and secure device-level access control.

HOMELAB

Hosted on an XCP-ng hypervisor with VLAN segmentation via pfSense and internal DNS. Features WireGuard tunnels, backup automation, and Git-based config management. Designed to test real-world production setups, integrate self-hosted tools, and refine deployment workflows.

  • Ubuntu Dev VM – Jenkins + automation scripts
  • Rocky Linux – Red Hat compatible services
  • Kali Linux – Ethical hacking toolsets
  • Prod Debian – Hardened NGINX + SMB + Docker
  • Pi on VLAN99 – WireGuard gateway
